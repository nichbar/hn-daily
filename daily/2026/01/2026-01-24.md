# 本期导读

周末的科技圈依然活跃。Claude Code 的新功能泄露引发了关于 AI “团队协作”的广泛讨论，而欧洲在打破通讯巨头壁垒方面迈出了实质性的一步。此外，从环境研究到项目管理的深度思考，本期内容涵盖了多个值得关注的硬核领域。

## 资讯

### Claude Code 泄露隐藏新功能：Swarms 多代理协作模式
[Claude Code](https://twitter.com/NicerInPerson/status/2014989679796347375) 最近被发现包含一个名为 “Swarms” 的隐藏模式。在该模式下，Claude 不再仅仅是编程助手，而是变身为“团队负责人”，负责规划和委派任务。一旦用户批准计划，它会产生一系列专业代理（如架构师、开发、测试等）并行工作。这种细粒度的分工旨在显著提升代码质量。评论区对此反响热烈，有开发者分享了使用 9 个由 Claude Opus 4.5 管理的代理进行复杂迁移的经验，认为分布式概念能更好地榨取 LLM 的性能。

### BirdyChat 成为首个与 WhatsApp 实现互操作性欧洲聊天应用
受益于欧盟《数字市场法案》（DMA），欧洲应用 [BirdyChat](https://www.birdy.chat/blog/first-to-interoperate-with-whatsapp) 宣布利用官方接口实现了与 WhatsApp 的消息互操作。EEA 区域的用户现在可以直接跨平台发送加密消息和文件。这一突破旨在解决新应用的“网络效应”壁垒。不过，评论指出 Meta 在实施时采取了“恶意合规”手段：WhatsApp 端需手动开启开关才能接收消息，这在很大程度上降低了实用性。

### 研究发现：电动汽车的普及显著降低了现实生活中的空气污染
南加州大学凯克医学院的一项[新研究](https://keck.usc.edu/news/adoption-of-electric-vehicles-tied-to-real-world-reductions-in-air-pollution-study-finds/)利用卫星数据证实，加州社区零排放车辆的增加与二氧化氮（NO2）水平下降直接相关。数据显示，每增加 200 辆电动车，NO2 水平下降 1.1%。这是首批利用现实世界数据证实电动车环境健康益处的研究。讨论中，虽然有人质疑电力来源的环保性，但普遍共识是集中发电比分散的内燃机排放更易控制。

### Comma openpilot：开源辅助驾驶系统的进击
开源辅助驾驶系统 [openpilot](https://comma.ai) 目前已支持超过 325 种车型。其最新硬件 comma four 宣称能提供优于车企原厂的辅助驾驶体验。该系统通过摄像头监控驾驶员注意力，允许更长时间的脱手驾驶，并能处理车道线丢失等复杂情况。用户反馈显示，其在雨雪天气或无标线道路上的表现远超原厂方案。

### 明尼苏达州再发联邦执法人员致命枪击事件引发抗议
明尼阿波利斯南部发生了一起涉及 [ICE/CBP 特工](https://www.startribune.com/ice-raids-minnesota/601546426)的致命枪击事件。受害者 Alex Pretti 是一名无犯罪记录的护士。尽管官方称其持械威胁，但现场视频及州调查机构被拒绝入场的情况引发了公众对执法透明度的强烈质疑，目前该事件已引发当地抗议。

## 博客

### 资深工程师谈如何评估工作量：估计本质上是政治工具
资深工程师 Sean Goedecke 在其[博文](https://www.seangoedecke.com/how-i-estimate-work/)中直言，准确评估软件耗时几乎是不可能的，因为开发本质上是探索未知。他认为“估计”在组织中常被作为管理层谈判的政治工具。他建议工程师应提供风险评估和多种替代方案，而不是死板的时间数字。评论区对此观点褒贬不一，讨论涉及了 SaaS 与硬件行业在截止日期文化上的巨大差异。

### 英国老房改造：利用旧电话线实现千兆以太网
针对缺乏以太网布线的旧宅，作者分享了通过 [G.hn 标准设备](https://thehftguy.com/2026/01/22/doing-gigabit-ethernet-over-my-british-phone-wires/)复用现有电话线实现千兆网络的方案。相比不稳定的电力猫，这种方式利用了电话线的专用频段，物理层带宽可达 1.7Gbps。读者补充道，许多 2000 年后的房屋其实预埋了 Cat5e 线缆，只需更换面板即可直接升级。
