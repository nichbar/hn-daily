# 本期导读

今天的技术圈充满了对“规则”的讨论与挑战。从德国电信对网络中立性的公然违反，到一加手机通过物理手段锁死系统降级，厂商与用户之间的博弈愈发激烈。而在学术界，一项被引用数千次的权威研究也被曝出存在严重缺陷，引发了对研究诚信的深刻反思。当然，极客精神依然闪光：有人在耳机里跑起了《毁灭战士》。

## 资讯

### 电动汽车普及正切实改善空气质量

[南加州大学的一项研究](https://keck.usc.edu/news/adoption-of-electric-vehicles-tied-to-real-world-reductions-in-air-pollution-study-finds/)为电动汽车的环保价值提供了实证支持。研究显示，随着加州零排放车辆比例的提升，当地因哮喘导致的急诊率显著下降。这一成果证明了交通工具的能源转型对公共卫生的直接益处，特别是在受污染影响最严重的贫困社区。

### 德国电信被指控违反网络中立性

多个消费者保护组织向监管机构申诉，指控 [Deutsche Telekom](https://netzbremse.de/en/) 人为制造网络瓶颈。申诉称，该运营商对未付费的服务商实施限速，导致 GitHub 和 Discord 等平台在高峰期难以访问。这种通过限制开放互联网来索取费用的行为，被批评者斥为一种“数字勒索”。

### 隐私危机：ICE 利用医疗补助数据进行监控

据 [EFF 报道](https://www.eff.org/deeplinks/2026/01/report-ice-using-palantir-tool-feeds-medicaid-data)，美国移民及海关执法局正在使用 Palantir 开发的工具，从 Medicaid 数据库中提取个人隐私信息。这意味着原本用于提供医疗保障的数据，正成为大规模监控和驱逐行动的利器，再次引发了公众对政府跨部门数据共享边界的担忧。

### 一加手机引入硬件级防回滚机制

一加在 [ColorOS 的最新更新](https://consumerrights.wiki/w/Oneplus_phone_update_introduces_hardware_anti-rollback)中采取了极端措施：通过烧毁处理器内部的电子熔断器（e-fuse），永久封堵了系统降级和刷入第三方 ROM 的路径。一旦用户尝试回滚，设备将彻底变砖。这一举动标志着一加从“极客友好”向封闭生态的彻底转型，引发了社区的强烈不满。

## 博客

### 学术诚信危机：被引用 6000 次的错误研究

学者 Andy King 在尝试复制一篇极具影响力的管理科学论文时，发现了其核心数据的[严重缺陷](https://statmodeling.stat.columbia.edu/2026/01/22/aking/)。令人唏嘘的是，作者和期刊在面对质疑时曾长期推诿，直到舆论爆发才发布勘误。这一事件揭示了学术界在激励机制和自我纠错功能上的制度性失败。

### 极客壮举：在蓝牙耳机上运行《毁灭战士》

开发者 Arin-s 成功将经典的 [DOOM](https://doombuds.com) 移植到了 Pinebuds Pro 耳机上。通过极限优化 1MB 的内存并大幅提升主频，他实现了视频流的流畅传输。这不仅是一次硬核的技术挑战，更是一份极具说服力的“求职作品”，证明了现代微控制器的惊人潜力。

### 写作秘籍：首先，要让读者产生兴趣

知名博主 Gwern 分享了他对[非虚构写作](https://gwern.net/blog/2026/make-me-care)的见解。他认为，好的文章不应开篇就罗列背景，而应通过展示知识鸿沟或揭示反直觉的现象来激起好奇心。在注意力稀缺的时代，让读者“在乎”是比单纯传递信息更优先的任务。

### 深入浅出：PostgreSQL 索引指南

对于希望优化数据库性能的开发者来说，这篇关于 [PostgreSQL 索引](https://dlt.github.io/blog/posts/introduction-to-postgresql-indexes/)的博文是极佳的参考。文章详细讲解了 B-Tree、GIN、BRIN 等多种索引的原理与适用场景，并深入探讨了覆盖索引和表达式索引等高级技巧。
